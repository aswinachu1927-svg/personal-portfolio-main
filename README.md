# Personal Portfolio

A sleek, modern, and high-performance personal portfolio built with HTML, CSS, and Vanilla JavaScript. Features a premium Apple-inspired aesthetic, glassmorphism UI elements, and incredibly smooth scroll-triggered animations.

## 🚀 Live Demo

**[View Live Project on Vercel](https://personal-portfolio-main-w34p.vercel.app/)**

## ✨ Key Features
- **Premium Design:** Modern dark mode aesthetic with vibrant color gradients.
- **Fluid Animations:** Parallax effects, interactive cursor glow, and background particle systems.
- **Smooth Scrolling:** Integrated with Lenis for a buttery-smooth scrolling experience.
- **Fully Responsive:** Optimized layouts for mobile, tablet, and desktop viewing.

## 🛠 Tech Stack
- HTML5
- CSS3 (Vanilla, Flexbox, CSS Grid)
- JavaScript (ES6+, DOM Manipulation, Intersection Observer API)
- Lenis Smooth Scroll
